# Warehouse-monitoring-system  仓库监控系统

## 项目介绍

This is a warehouse monitoring system, by using springboot, vue, mybatis-plus, etc.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
